# krkrz 编译说明

> 原文：`engine/HowToBulid.md`（英文 QuickStart + 中文详细说明，已原位 UTF-8 汉化）  
> 面向需要从源码编译 **吉里吉里 Z（krkrz）** 本体或插件的开发者。  
> 日常制作 Gal 只需 `runtime/win32/krkrz.exe`，**不必编译**本目录。

---

## 快速编译（最小步骤）

编译吉里吉里 Z 需要：

- **Visual Studio 2012 Update 4** 或更高版本
- **nasm 2.10.09**（是否兼容更新版本尚未验证）  
  从 http://www.nasm.us/ 下载，并将 `nasm.exe` 所在目录加入环境变量 `PATH`

编译步骤：

1. 用 Visual Studio 打开 `engine/src/core/vc2012/tvpwin32.sln`
2. 生成解决方案中的 **tvpwin32** 项目
3. 输出二进制：`engine/bin/win32/tvpwin32(_d).exe`（Debug 带 `_d` 后缀）

## 自动生成文件

吉里吉里 Z 本体含若干 **自动生成** 的源码文件。  
**不要直接编辑生成结果**，应修改 **生成源**，再运行对应脚本重新生成。  
生成脚本主要使用 **bat** 与 **Perl**，因此需安装 Perl。

下列列出 **生成结果 ← 生成源**（`:` 左侧为输出，右侧为输入）。

### TJS 语法解析（bison）

由 `src/core/tjs2/syntax/compile.bat` 生成：

| 生成文件 | 源文件 |
|----------|--------|
| `tjs.tab.cpp` / `tjs.tab.hpp` | `tjs.y` |
| `tjsdate.tab.cpp` / `tjsdate.tab.hpp` | `tjsdate.y` |
| `tjspp.tab.cpp` / `tjspp.tab.hpp` | `tjspp.y` |
| `tjsDateWordMap.cc` | `gen_wordtable.bat` |

需要 **bison**。bison 运行还依赖以下 DLL，需一并安装：

- libiconv2.dll  
- libintl3.dll  
- regex2.dll  

参考：

- http://gnuwin32.sourceforge.net/packages/bison.htm  
- http://gnuwin32.sourceforge.net/packages/libintl.htm  
- http://gnuwin32.sourceforge.net/packages/libiconv.htm  
- http://gnuwin32.sourceforge.net/packages/regex.htm  

### OpenGL 接口表

由 `src/core/visual/glgen/gengl.bat` 生成：

| 生成文件 | 源文件 |
|----------|--------|
| `tvpgl.c` / `tvpgl.h` | `maketab.c` / `tvpps.c` |

### IA32 图形例程

由 `src/core/visual/IA32/compile.bat` 生成：

| 生成文件 | 源文件 |
|----------|--------|
| `tvpgl_ia32_intf.c` / `tvpgl_ia32_intf.h` | `*.nas` |

### 引擎选项描述（`--userconf` 界面）

引擎设置界面文案以 **JSON 资源** 嵌入 exe，**不要**再使用旧版 `option_desc_ja.txt` 路径。

| 项目 | 路径 / 说明 |
|------|-------------|
| 编辑源文件 | `engine/src/core/vcproj/option_desc_ja.json`（本仓库已汉化，文件名保留以兼容构建） |
| 资源 ID | `IDR_OPTION_DESC_JSON`（TEXT 类型） |
| 运行时 | `krkrz.exe --userconf` 读取上述资源 |

修改 JSON 后需 **重新编译 tvpwin32** 才会进入 exe。详见 [引擎选项说明.md](引擎选项说明.md)。

### 导出桩（插件接口）

由 `src/core/base/win32/makestub.bat` 生成：

| 生成文件 | 源 |
|----------|-----|
| `FuncStubs.cpp` / `FuncStubs.h` | `makestub.pl` 所指定头文件中，由 `TJS_EXP_FUNC_DEF` / `TVP_GL_FUNC_PTR_EXTERN_DECL` 宏列出的函数 |
| `tp_stub.cpp` / `tp_stub.h` | 同上 |

## 编译插件

官方 Win32 插件源码位于 `engine/src/plugins/win32/`（KAGParser、wuvorbis、menu 等）。  
使用与本体相同的 Visual Studio 环境，打开对应 `.sln` / 项目生成 DLL，复制到 `runtime/win32/plugin/` 测试。

本仓库已预置的 7 个插件见 [runtime/README.md](../runtime/README.md)。

---

## 与本仓库的关系

| 需求 | 做法 |
|------|------|
| 运行 Gal / 改 KAG 剧本 | 使用 `runtime/win32/krkrz.exe`，运行 `ops/bootstrap-runtime.ps1` |
| 修改引擎或编译新插件 | 初始化 `engine` 子模块后按本文编译 |
| 替换运行时 | 将生成的 `tvpwin32.exe` 复制为 `runtime/win32/krkrz.exe`（注意 GPL 合规） |
