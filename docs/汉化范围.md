## 编码与维护

- 中文写入 `engine/` 建议经 `_encode_staging/` → UTF-8 BOM 部署（见 `engine/GAL-README.md`）。
- **勿** 在未备份时对 `engine/` 执行 `git submodule update --recursive`，以免覆盖汉化。
- **勿** 修改 `engine/script/KAG3`、`KAG3_Ham` 内的占位 txt 或 system 脚本；玩家向占位说明与汉化 KAG 仅维护 `projects/_template/data/`。
- 验收：`ops/verify-krkr.ps1`（含 BOM、关键汉化文件、KAG3 参考子模块洁净性检查）。
