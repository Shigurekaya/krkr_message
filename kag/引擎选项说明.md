# krkrz 引擎选项（userconf / option_desc）

> 原文：`engine/document/misc_markdown/user/engine_setting.md`、`core/engine_setting.md`（已汉化，见 `engine/document/misc_markdown/`）  
> 面向需要调试 **krkrz.exe 引擎参数** 的开发者；与游戏内 KAG `Config.tjs` 不同。

---

## 用户向引擎设置

吉里吉里 Z **不再提供** 独立的「引擎设置工具」程序。

以 **`--userconf`** 参数启动本体时，会进入 **面向玩家的引擎设置界面**：

```text
krkrz.exe --userconf
```

可在安装包或快捷方式中为 `krkrz.exe` 加上 `--userconf`，效果等同旧版单独的设置工具。

---

## 开发者向设置

**官方不提供** 单独的「开发者用引擎设置工具」。

若希望某些选项 **不对玩家显示**：

- 在 `option_desc_ja.json` 中将对应项设为 `"user": false`，或
- 直接从 JSON 中删除该项。

设置界面异常时，可能是 JSON 语法错误；建议 **从控制台启动** 查看日志。

修改 `option_desc_ja.json` 后需 **重新编译 core** 或使用可替换资源的工具，变更才会进入 `krkrz.exe`。

JSON 文件路径（子模块内）：

```text
engine/src/core/vcproj/option_desc_ja.json
```

界面文案源文件 `option_desc_ja.json` 在本仓库中已 **汉化**（文件名保留以兼容构建与插件资源 ID）。

## 与吉里吉里 2 的差异（概要）

| 时代 | 方式 |
|------|------|
| krkr2 | 本体通过 `TVPGetCommandDesc` 等获取 `\|` 分隔的描述字符串；插件 DLL 导出 `GetOptionDesc` |
| krkrz | 选项列表改为 **JSON**，作为资源嵌入；本体使用 `option_desc_ja.json` |

插件若需注册选项：在 DLL 资源中加入 `TEXT` 类型、`IDR_OPTION_DESC_JSON` 指向 `option_desc_ja.json`（JSON 字段说明见 [engine/document/misc_markdown/core/engine_setting.md](../engine/document/misc_markdown/core/engine_setting.md)）。

汉化范围总览见 [docs/汉化范围.md](../docs/汉化范围.md)。

---

## 与 KAG Config.tjs 的区别

| 项目 | 引擎选项（userconf） | KAG Config.tjs |
|------|----------------------|----------------|
| 作用对象 | krkrz 引擎（数据路径、内存、图形缓存等） | 游戏 system（分辨率、存档、菜单、对话框） |
| 配置文件 | `option_desc_ja.json` + 引擎内置存储 | `projects/<名>/data/system/Config.tjs` |
| 本仓库文档 | 本文 | [Config-常用项.md](Config-常用项.md) |
