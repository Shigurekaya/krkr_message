## 编码与维护

- 中文写入须 UTF-8 BOM（`utf-8-sig` / `UTF8Encoding($true)`）。
- GAL 新项目：**禁止**依赖 Shift_JIS；默认 `readencoding` / `TJS_mbstowcs` / `TextStream` 均为 **UTF-8**。
- `engine/` 已 vendorize（普通目录）；**勿**再执行 `git submodule update`；可从 `cn-applied/tree` 恢复汉化成品。
- 玩家向汉化以 `projects/_template/data/` 与 `shared/gal-settings-ui/` 为准。
- 验收：`ops/verify-krkr.ps1`；进度见 gal- `docs/pre-l10n-backup/l10n-status.md`。
- **下一刀**：引擎 / 插件 C++ 日文注释（人工/AI，勿机翻脚本）。
