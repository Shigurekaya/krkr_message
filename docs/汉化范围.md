## 编码与维护

- 中文写入须 UTF-8 BOM（`utf-8-sig` / `UTF8Encoding($true)`）。
- GAL 新项目：**禁止**以 Shift_JIS 作为项目编码；仅当打开旧日文资源时才用 `-readencoding=Shift_JIS`。
- **勿** 在未备份时对 `engine/` 执行 `git submodule update --recursive`，以免覆盖汉化；可从 `cn-applied/tree` 恢复。
- 玩家向汉化以 `projects/_template/data/` 为准；engine 内 KAG3 参考树可同步，运行不依赖它。
- 验收：`ops/verify-krkr.ps1`；进度见 gal- `docs/pre-l10n-backup/l10n-status.md`。
